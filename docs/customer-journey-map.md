## 重要インサイト：年収ではなく「生涯収入×やりがい」で訴求

### 大手企業50代後半の年収推移（現実）

```
55歳前（管理職）:  800万〜1,200万円
  ↓ 役職定年で ▲20-30%
57歳（役職定年後）: 600万〜900万円
  ↓ 再雇用で ▲40-50%
60歳（再雇用）:    300万〜500万円
```

### 「年収が下がってでも行きたい理由」の作り方

**単年の年収比較では資格取得のメリットが見えにくい。** 大企業の再雇用（300-500万）とブルーカラー資格後（350-700万）はほぼ同水準。

**訴求軸を2つに分ける:**

**訴求1: 生涯収入の逆転（数字で納得させる）**

| | 再雇用コース | BWD資格取得コース |
|--|--|--|
| 60-65歳 | 400万×5年 = 2,000万 | 450万×5年 = 2,250万 |
| 65-70歳 | パート150万×5年 = 750万 | 400万×5年 = 2,000万 |
| 70-75歳 | ほぼゼロ | 300万×3年 = 900万 |
| **生涯収入合計** | **約2,750万円** | **約5,150万円（+2,400万円）** |

**訴求2: やりがい・自己効力感（感情で動かす）**

- 「再雇用で居場所なく座っている5年間」 vs 「自分の手で仕事をする充実した10年間」
- 「65歳で強制終了」 vs 「自分で引退時期を決められる」
- 「会社にしがみつく」 vs 「手に職で自立する」

### プロダクトへの反映

- 「推定年収」表示 → **「生涯収入シミュレーション」** に変更
- 再雇用コースとの比較グラフを自動生成
- 資格取得者のストーリー（「手に職の充実感」）を前面に
