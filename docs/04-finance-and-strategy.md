## 1. 経営戦略

### 1.1 競争戦略: 「ブルーカラー × AI × 無料」のポジショニング

BWDの競争優位性は、以下の3つの掛け合わせにある:

```
          高額
           ↑
  ユーキャン  │  職業訓練校
  スタディング │  ポリテク
           │
  ─────────┼──────────→ ブルーカラー特化
           │
  YouTube  │  ★ BWD ★
  独学     │  (無料×AI×特化)
           ↓
          無料
```

**なぜ真似されにくいか:**
1. **データの壁**: ユーザーの学習データが蓄積されるほどAIの精度が上がる（ネットワーク効果）
2. **コンテンツの壁**: 資格ごとの体系的カリキュラムの構築には時間がかかる
3. **信頼の壁**: 合格実績・企業導入実績が積み上がるほど参入障壁になる

### 1.2 事業ポートフォリオ

```
収益の柱:
┌──────────────────────────────────────────┐
│  ① B2B SaaS（安定収益・MRR）            │ → 全体の40%
│  ② 人材紹介（高単価・成長ドライバー）      │ → 全体の50%
│  ③ 広告・提携（副収益）                  │ → 全体の10%
└──────────────────────────────────────────┘
```

### 1.3 参入戦略: 電気工事士から始める

17以上の資格に対応しているが、初期は**電気工事士（第二種）に集中**する。

**理由:**
- 需要が「極めて高い」（再エネ・EV需要）
- 試験が年2回あり、合格実績を作りやすい
- 合格後の年収インパクトが大きい（320万→480万円）
- コンテンツが最も充実している（5レッスン+クイズ完備）

**電気工事士で成功モデルを確立 → 他の資格に横展開**

### 2.2 損益分岐点分析

**Phase 1（2026年）の損益分岐点:**
- 月間固定費: 約6万円
- プロプラン1社で月29,800円 → **2社契約で黒字化**
- 極めて低いBEP（損益分岐点）

**Phase 2（2027年）の損益分岐点:**
- 月間固定費: 約180万円（人件費含む）
- プロプラン60社 or エンタープライズ18社で黒字化
- 人材紹介が加わればさらに早期に黒字化

### 2.3 資金調達戦略

#### Phase 1: ブートストラップ（2026年）
- **自己資金のみ**: 44万円/年で運営可能
- オーナーの他事業（Canal AI）からの持ち出し
- 外部調達不要

#### Phase 2-3: ブートストラップ経営（オーナー決定: 2026年4月15日）
- **方針:** 外部資金調達は行わず、売上利益を再投資して成長する
- **メリット:**
  - 株式希薄化なし、オーナーが100%の意思決定権を保持
  - 投資家への報告義務なし、自由な経営判断が可能
  - 黒字経営を前提とした堅実な成長
- **リスク対策:**
  - Phase 1のコスト構造（月6万円）を維持し、固定費を最小化
  - 人材採用は売上が安定してから段階的に実施
  - 必要に応じて日本政策金融公庫の創業融資（低利）を検討

## 3. 法人設立計画

### 3.1 推奨タイミング: 2026年8月

**理由:**
- 初の合格者輩出で実績ができる
- B2B契約には法人格が有利
- 人材紹介業の許認可申請には法人が必要

### 3.2 法人形態: 合同会社（LLC）

**理由:**
- 設立費用が安い（約6万円 vs 株式会社の約25万円）
- 設立手続きが簡易
- 後で株式会社への組織変更が可能
- VCからの調達を検討する段階で株式会社に変更

### 3.3 設立時の必要事項
| 項目 | 内容 | 費用 |
|------|------|------|
| 定款作成 | 電子定款（収入印紙不要） | 0円 |
| 登録免許税 | 法務局への登記 | 6万円 |
| 法人印鑑 | 実印・銀行印・角印 | 1万円 |
| 法人口座開設 | ネット銀行推奨 | 0円 |
| 税理士（顧問） | 決算期のみ依頼 | 年10万円 |
| **合計** | | **約17万円** |

## 5. リスク管理

### 5.1 事業リスク

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|---------|------|
| PMFが見つからない | 高 | 中 | 2026年Q3に判定。見つからなければB2B特化にピボット |
| 競合の参入 | 中 | 中 | データ蓄積とコンテンツ品質で先行者優位を確保 |
| AI APIコスト増 | 中 | 低 | 複数AIプロバイダー対応（Gemini→Claude切替可能に） |
| 法規制の変更 | 中 | 低 | 顧問弁護士の確保（Phase 2） |
| 個人情報漏洩 | 高 | 低 | Supabase RLS、暗号化、定期セキュリティ監査 |

### 5.2 撤退基準（オーナー決定: 2026年4月15日）

**2026年12月末時点で、登録ユーザーが100名を超えない場合、事業を撤退する。**

この基準はシンプルかつ明確であり、判断に迷いが生じない。
