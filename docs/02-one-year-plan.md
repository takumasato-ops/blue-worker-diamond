## 事業コンセプト

### 「大手企業の50代社員に、セカンドキャリアの選択肢を無償で届ける」

日本の大手企業は50代社員のセカンドキャリア支援に頭を抱えている。
役職定年、早期退職、定年延長——いずれのケースでも「次のキャリアをどう描くか」が課題になる。

BWDは**大手企業に対して、社員のセカンドキャリア支援ツールとして無償提供**する。
企業側の負担はゼロ。社員は無料でブルーカラー国家資格を取得し、第二の人生を切り拓く。

**「ホワイトカラーから、手に職のプロフェッショナルへ。」**

---

## なぜこの戦略が刺さるか

### 大手企業の課題

| 課題 | 詳細 |
|------|------|
| **50代社員の滞留** | ポスト不足で管理職になれない50代が増加。モチベーション低下が組織全体に影響 |
| **役職定年後の処遇** | 55歳前後で役職を外れた社員のキャリアパスが不明確 |
| **早期退職制度の限界** | 退職金を積んでも「辞めた後どうするか」がないと応募されない |
| **70歳雇用義務化** | 2025年4月から努力義務。65歳以降の就業機会確保が経営課題 |
| **ESG・人的資本開示** | 人材投資の可視化が求められ、セカンドキャリア支援は開示項目になりつつある |

### BWDが提供する価値

| 企業にとって | 社員にとって |
|------------|------------|
| セカンドキャリア支援制度を**コストゼロ**で導入できる | 無料でAI家庭教師つきの資格学習ができる |
| 人的資本開示に「セカンドキャリア支援」として記載できる | 「手に職」という具体的な選択肢が見える |
| 社員のモチベーション維持・離職防止につながる | 市場価値が数字で可視化され、自信がつく |
| 退職後も活躍する社員の実績が企業ブランドになる | 資格取得→転職・独立という明確なルートがある |

### 市場規模

- 従業員1,000人以上の大手企業: 約4,000社
- 50代社員の割合: 約20-25%
- セカンドキャリア支援に予算を持つ企業: 増加中（人的資本開示の影響）
- **1社あたり50-200人の50代社員 × 4,000社 = 数十万人規模の潜在ユーザー**

## 基本方針

### お金をかけない。大手企業の「困りごと」を解決する。

1. **大手企業に無償提供** — BWDのProプラン相当を無料で使ってもらう
2. **収益は人材紹介で回収** — 資格取得した50代社員を、人手不足の建設・電気業界にマッチング
3. **公的機関も並行活用** — ハローワーク、自治体を通じたリーチも維持

### 収益モデル

```
大手企業が無償導入 → 50代社員が資格取得 → 転職希望者をマッチング → 紹介手数料で収益

企業からの収益:  ¥0（無償提供）
人材紹介収益:    年収400万×25% = 1件100万円
```

**1社から月1名の紹介成約で月100万円。年間で10社導入すれば年商1,000万円以上。**

## 提案の切り口

### 大手企業の人事部門への提案

**件名:** 「50代社員のセカンドキャリア支援を、コストゼロで始めませんか」

**提案骨子:**

> 御社の50代社員向けに、AI資格学習プラットフォーム「Blue Worker Diamond」を
> **完全無償**でご提供します。
>
> **BWDでできること:**
> - 6学科・17国家資格のオンライン学習（AI家庭教師つき）
> - 一人ひとりの適性に合った資格・キャリアをAIが診断
> - 学習進捗と市場価値の可視化
> - 資格取得後の転職マッチング支援
>
> **御社のメリット:**
> - 導入コスト・運用コスト: ゼロ
> - 人的資本開示に「セカンドキャリア支援制度」として記載可能
> - 社員のモチベーション維持、組織活性化
> - 早期退職制度と組み合わせで「辞めた後の道」を提示
>
> **実績:**
> - 合格者〇〇名（AI学習による平均学習期間3ヶ月）
> - ハローワーク〇〇箇所と連携中
>
> まずは10名程度のトライアルからいかがでしょうか。
