**テーマ:** 「収益化と組織化」

| MAU | 8,000人 |
| 資格合格者数（累計） | 2,000人 |
| B2B導入企業 | 100社 |
| MRR | 300万円 |
| 人材紹介成約 | 月5件 |
| ARR（年間定期収益） | 3,600万円 |
| 従業員数 | 5名 |

### 重点施策
1. **収益の多角化**: B2B SaaS + 人材紹介の2本柱を確立
2. **プロダクト強化**: 管理ダッシュボード、模擬試験、PWA対応
3. **チーム構築**: エンジニア1名、営業1名、CS1名を採用
4. **ブートストラップ経営**: 外部調達せず、売上利益を再投資して成長（オーナー決定）
5. **PR本格化**: メディア露出、カンファレンス登壇、成功事例の発信

## リスクシナリオと対策

### 撤退基準（オーナー決定: 2026年4月15日）
**2026年12月末時点で登録ユーザーが100名を超えない場合、事業を撤退する。**

### 最悪ケース: PMFが見つからない（2026年Q3時点で登録50人未満）
- **対策:** ターゲットを絞る（例: 電気工事士特化）、B2Cを縮小しB2B特化にピボット

### 中間ケース: 成長が遅い（2027年末でMRR 100万円未満）
- **対策:** 人材紹介収益を先行させ、プロダクト投資を継続。外部資金に依存しないブートストラップ体質を維持

### 最良ケース: 急成長（2027年Q2でMRR 300万円到達）
- **対策:** 利益を再投資し、自己資金でプロダクト開発と営業体制を拡大
