## 1. マーケティング全体戦略

### 1.1 ポジショニング

**一言で言うと:**
> 「ブルーカラー版スタディサプリ。ただし、AIが家庭教師で、完全無料。」

**キーメッセージ:**
- 求職者向け: 「手に職をつけたいなら、BWD。無料で国家資格が取れる。」
- 企業向け: 「社員の資格取得を、AIで半分の期間に。」

### 1.2 ファネル設計

```
認知 → 興味 → 登録 → 学習開始 → 資格合格 → （企業マッチング）
  ↓      ↓      ↓       ↓          ↓            ↓
 SEO    LP    無料    AI家庭教師   実績化       人材紹介
 SNS   動画   診断    動画+クイズ  体験記       収益化
 PR    広告   相談    弱点分析    口コミ       B2B提案
```

### 1.3 ターゲットセグメント別アプローチ

| セグメント | チャネル | メッセージ | CTA |
|-----------|---------|----------|-----|
| 20-30代 転職検討者 | SNS（X, TikTok）, YouTube | 「AI時代に食いっぱぐれない資格」 | 無料診断 |
| 30-40代 年収UP希望 | SEO, Google広告 | 「3ヶ月で年収+150万円」 | 学科一覧 |
| 40-50代 セカンドキャリア | ハローワーク, SEO | 「40代からでも遅くない」 | 卒業生の声 |
| 企業 人事担当 | LinkedIn, メール営業, 展示会 | 「教育コスト60%削減」 | 導入相談 |

### 2.2 SNS戦略

- **投稿頻度:** 毎日1-2投稿
- **コンテンツミックス:**

  - オーナーの想い・ストーリー（10%）
- **目標:** 2026年末 フォロワー1,000人

- 「3分でわかる○○資格」シリーズ
- 合格者インタビュー

### 2.3 広告戦略

#### Phase 1（2026年5-12月）: テスト期
- **予算:** 月2-3万円
- **チャネル:** Google検索広告、X広告
- **目的:** CPA（獲得単価）の測定、勝ちクリエイティブの発見

**Google検索広告:**
- キーワード: 「電気工事士 資格 無料」「フォークリフト 資格 取り方」
- ランディングページ: /jobs（学科一覧）or /onboarding（AI診断）

**X広告:**

- 目標CPA: 300円以下

- **予算:** 月30万円
- **チャネル追加:** YouTube広告、LinkedIn広告（B2B）、リスティング広告拡大
- **リターゲティング:** サイト訪問者への再アプローチ

## 4. B2B営業戦略

### 4.1 ターゲット企業

**Tier 1（最優先）: 中堅建設・電気工事会社**
- 従業員50-500人
- 資格取得ニーズが高い
- 教育予算がある
- アプローチ: メール営業 + オンラインデモ

**Tier 2: 物流・製造業**
- フォークリフト・危険物の資格ニーズ
- 教育担当者が限られている
- アプローチ: SEO経由の問い合わせ対応

**Tier 3: ビルメンテナンス会社**
- 複数資格（4点セット）の取得ニーズ
- 社員定着率の課題
- アプローチ: Phase 2以降で展開

### 4.2 営業プロセス

```
リスト作成 → メール → 返信 → デモ → 無料トライアル → 契約
  (AI CEO)   (AI CEO)  (3日以内)  (30分)  (1ヶ月)      (プロプラン)
```

**目標パイプライン（月次）:**
- メール送信: 40社/月
- 返信率: 10%（4社）
- デモ実施: 3社/月
- トライアル開始: 2社/月
- 契約: 1社/月

### 4.3 セールスツール

| ツール | 内容 | ステータス |
|--------|------|----------|
| セールスデック（PDF） | BWD紹介、導入メリット、料金、事例 | 7月作成 |

## 6. ブランドガイドライン

### 6.1 トーン&マナー

| 要素 | ガイドライン |
|------|------------|
| **口調** | 親しみやすく、でもプロフェッショナル。「です・ます」調 |
| **人称** | ユーザーは「あなた」、BWDは「私たち」 |
| **避ける表現** | 「底辺」「ブラック」「誰でもできる」など、職業を蔑む表現は絶対に使わない |
| **推奨表現** | 「手に職」「プロフェッショナル」「現場のプロ」「ダイヤモンド」 |
| **数字の使い方** | 過大広告にならない。実績は実データに基づく。「最大」「約」を適切に使用 |
