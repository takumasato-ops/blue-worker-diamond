## 運用ルール

1. **AI CEO少年は毎日16時までに「日次進捗ログ」に記録**
2. **ステータス変更はリアルタイムで更新**
3. **ブロッカーがある場合は即座にオーナーに報告**
4. **月末に月次KPIダッシュボードを更新**
5. **撤退基準（登録100名）への進捗を常に可視化**
