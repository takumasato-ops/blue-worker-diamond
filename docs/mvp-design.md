# BWD MVP設計書

## 生涯収入シミュレーション設計

### ユーザー入力

| 項目 | 例 |
|--|--|
| 現在の年齢 | 56歳 |
| 現在の年収 | 800万円 |
| 希望する働き方 | 70歳まで現役 / 65歳で引退 |

### 計算ロジック

**変数:**

| 変数 | データソース |
|--|--|
| 再雇用の年収推移 | 厚労省 賃金構造基本統計調査 |
| 資格別の平均年収 | 求人サイトの集計データ |
| 年齢別就業率 | 総務省 労働力調査 |
| 資格の組み合わせボーナス | 複数資格で+10-20%の補正 |

**出力例:**

| | 再雇用コース | BWD資格取得コース |
|--|--|--|
| 56-60歳 | 700万×4年 = 2,800万 | 700万×4年 = 2,800万（学習中） |
| 60-65歳 | 400万×5年 = 2,000万 | 480万×5年 = 2,400万 |
| 65-70歳 | 150万×5年 = 750万 | 400万×5年 = 2,000万 |
| 70-75歳 | 0円 | 250万×3年 = 750万 |
| **合計** | **約5,550万** | **約7,950万（+2,400万）** |

## 営業トーク・LP改修方向性

### 新LP構成（10セクション）

1. **ヒーロー:** 「御社の50代社員の生涯収入を、2,400万円増やしませんか。」
2. **課題:** 役職定年→再雇用→65歳で終了の構造的問題
3. **生涯収入グラフ:** 再雇用コース vs 資格取得コースの比較
4. **やりがい訴求:** 「居場所のない再雇用」vs「手に職で生涯現役」
5. **BWDの仕組み:** AI適性診断→AI家庭教師→資格取得→セカンドキャリア
6. **法人メリット:** 管理ダッシュボード/人的資本開示対応/進捗可視化
7. **価格+助成金:** 月額20万〜/助成金で最大75%OFF
8. **導入フロー:** 問い合わせ→トライアル10名→全社展開
9. **FAQ:** 生涯収入シミュレーションの根拠等
10. **CTA:** 「まずは10名のトライアルから」

### キラーフレーズ

**人事向け:** 「再雇用制度だけだと65歳で終わりです。BWDなら社員が70歳以降も自立して働ける力を、御社の在籍中に身につけられます。生涯収入で見ると2,400万円の差が出ます」

**経営層向け:** 「早期退職制度に退職金を積んでも応募が集まらないのは、辞めた後の出口がないからです。BWDがその出口になります。しかも助成金で最大75%OFF」

## 助成金スキーム

### 対象助成金

| 助成金 | 助成率（中小） | 助成率（大企業） | 上限 |
|--|--|--|--|
| 人材育成支援コース | 45%（最大60%） | 30%（最大45%） | 1人15万〜50万円 |
| 人への投資促進コース | 最大75% | 最大60% | 1人15万〜50万円 |
| 事業展開等リスキリング支援コース | 最大75% | 最大60% | 1人15万〜50万円 |

### 申請フロー

1. 企業がBWDを契約
2. BWDが「訓練計画書」のテンプレを提供
3. 企業の社労士（提携 or 顧問）が計画届を労働局へ提出（訓練開始日の1ヶ月前まで）
4. 従業員がBWDで学習開始（10時間以上が必須条件）
5. BWDが学習履歴・修了証を自動出力（助成金申請の証拠書類）
6. 社労士が支給申請 → 助成金が企業に入金

### BWDが提供する助成金関連機能

- 訓練計画書テンプレ
- 学習履歴レポート（1人ごとの学習時間・内容の記録）
- 修了証の自動発行
- 助成金ガイド資料

### 注意書き（LP・契約書に明記）

> ※ 助成金の申請は企業の提携社労士または顧問社労士が行います。BWDは申請に必要な学習データ・証拠書類を提供しますが、助成金の受給を保証するものではありません。

## ターゲット企業

### 初期ターゲット条件

- 従業員1,000人以上
- 役職定年制度あり
- 人的資本開示に積極的
- 製造・インフラ系

### 優先業界

| 優先度 | 業界 | 代表企業 |
|--|--|--|
| 1 | 電機メーカー | 日立、パナソニック、三菱電機、東芝 |
| 2 | 建設・ゼネコン | 大林組、鹿島、清水建設、竹中工務店 |

| 5 | メガバンク・保険 | 三菱UFJ、みずほ、三井住友 |

### アプローチ先

人事部 → 人材開発課 / キャリア開発室 / ダイバーシティ推進室

### 営業フェーズ

| フェーズ | 時期 | 方法 |
|--|--|--|
| 1 | 5月 | 人脈・紹介（10名無料トライアル提案） |

### 初回提案の武器

- 生涯収入シミュレーションのデモ

## 画面構成（6画面）

1. **ログイン/招待受付** — 企業からの招待メールでアカウント作成
2. **AI適性診断（初回）** — 4問の質問→3つの資格を推薦→生涯収入シミュレーション表示
3. **生涯収入シミュレーション** — 再雇用 vs 資格取得の比較グラフ+差額表示
4. **学習画面（メイン）** — 過去問+進捗バー+AI家庭教師への質問ボタン
5. **AI家庭教師チャット** — Claude APIベースのチャットUI。図解対応
6. **法人管理ダッシュボード（人事向け）** — 利用者一覧/進捗/合格率/レポートDL/助成金書類DL
